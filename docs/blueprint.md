# **App Name**: Rust365

## Core Features:

- AI-Powered Visual Assets: Utilize generative AI to create a unique hero background image and brand logo, storing these optimized assets on Firebase Storage for consistent, cost-effective display across the platform.
- User Authentication & Profiles: Secure user registration, login, and basic profile management (e.g., username, profile picture) using Firebase Authentication and Firestore for storing user-specific data.
- Dynamic Rust Server Directory: Display a sortable and filterable list of Rust game servers by population, region, and game mode, providing real-time status updates.
- Community Message Board: A feature-rich message board allowing authenticated users to create new topics, post replies, and engage in discussions relevant to the Rust community.
- Curated Rust News Hub: A dedicated section showcasing the latest news, updates, and developer announcements from the world of Rust, updated regularly.
- Featured Content Creator Showcase: Highlight prominent Rust content creators, featuring their latest videos, streams, or guides with direct links.

## Style Guidelines:

- Primary interactive and accent color: A bold, vibrant red (#FA0D1B) to evoke intensity and energy, standing out effectively against dark backgrounds.
- Background color: A very dark charcoal with subtle red undertones (#1C1919), providing a deep, immersive foundation for the application's aesthetic.
- Secondary accent color: A lighter, desaturated purplish-pink (#D0A9BF) for subtle highlights or secondary UI elements, offering visual differentiation while remaining analogous to the primary red.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy, and slightly rugged feel, complementing the game's aesthetic.
- Body font: 'Inter' (sans-serif) for clean readability and neutrality across all content, particularly for message board posts and news articles.
- Use clean, modern vector icons that subtly reference Rust's survival and industrial themes, ensuring clarity against dark backgrounds.
- Employ a modular, card-based layout for servers, news, and creator features, maximizing content density while maintaining readability with generous dark spacing. A prominent hero section will showcase the AI-generated visual assets.
- Implement subtle, fast-response animations for hover states and button interactions, enhancing the dynamic feel without distracting from the content. Smooth transitions for page navigation to improve user flow.
